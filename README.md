
AverageFaceDemo
===================================
Android平均脸合成应用。
  
基于 
----------------------------------- 
* [dlib-android](https://github.com/tzutalin/dlib-android) 提供Android平台可用的Dlib库。
* [LearnOpenCV.com](https://github.com/spmallick/learnopencv/tree/master/FaceAverage) 的平均脸合成算法。
